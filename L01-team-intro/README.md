## Overview

Lab 01 was the team's first presentation for ITAI 4376, introducing Neural Nexus and the CareerKiosk project vision.

## What We Presented

### Team Introduction
- Meet Neural Nexus: 5 AI students from HCC
- Individual backgrounds, skills, and expertise areas
- Assigned roles for the CareerKiosk project

### Project Vision: CareerKiosk
- **Problem:** Job seekers and employers lack an AI-powered platform for modern career navigation
- **Solution:** A multi-portal, AI-driven career marketplace
- **Target Users:** Job seekers, employers, career experts, and institutions

### Proposed Features
- AI-powered resume builder and optimization
- Smart job-to-candidate matching
- AI interview coach with mock sessions
- Multi-portal architecture (Job Seeker, Company, Admin)
- Mobile-first responsive design

### Technology Stack
- React + TypeScript + Tailwind CSS frontend
- Supabase backend with PostgreSQL
- Google Gemini API for AI features
- Lovable AI for rapid development

## Outcome

Project approved. Team proceeded with full CareerKiosk development, delivering a production-ready platform by end of semester.
