## 🌐 Flagship Project: CareerKiosk

**CareerKiosk** is a professional AI-powered career guidance marketplace that connects job seekers with employers, career experts, and institutions. Built from scratch over one semester at Houston Community College, it grew into a fully functional, production-ready platform presented at the **HCC AI Innovation Symposium & Expo — Spring 2026**.

🔗 **Live Platform:** [careerkiosk.ai](https://careerkiosk.ai)

### Platform Highlights
- **130+ screens** across 3 live portals (5 portals total planned)
- **50+ database tables** with Row-Level Security on every table
- **28 serverless Edge Functions** for backend logic
- **95+ database migrations** throughout development
- **10+ AI-powered features** across all portals
- **Mobile-ready** via PWA + Capacitor (iOS & Android)
- **Enterprise-grade security:** 2FA, Google OAuth, RLS, database triggers, audit logs
